# e-plantShopping

e-plantShopping is a simple e-commerce web application for browsing and purchasing plants online. It demonstrates a modern front-end architecture with state management and cart functionality.

This project is part of the Coursera course: [Developing Front-End Apps with React](https://www.coursera.org/learn/developing-frontend-apps-with-react/home/welcome)

## Features

- Browse a catalog of plants
- Add and remove items from cart
- Dynamic cart total calculation
- State management with Redux
- Responsive UI

## Tech Stack

- React
- Redux Toolkit
- JavaScript (ES6+)
- HTML5 & CSS3
- Vite

## Getting Started

### 1. Clone the repository

git clone https://github.com/jertol/e-plantShopping.git  
cd e-plantShopping

### 2. Install dependencies

npm install

### 3. Run the development server

npm run preview

The app will be available at the local development URL shown in your terminal.

## Project Structure

- `src/` – Application source code  
- `components/` – Reusable UI components  
- `redux/` – Redux store and slices  
- `assets/` – Static assets
